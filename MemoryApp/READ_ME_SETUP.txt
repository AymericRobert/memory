
SETUP de l'application MEMORY :

-Ensuite, mettre son fichier dans le dossier de xampp/htdocs ( par default dans c: )
-aller a l'url : http://localhost/MemoryApp/FrontEnd/index.html.

-Pour creer sa database : 
aller a l'url http://localhost et phpAdmin
